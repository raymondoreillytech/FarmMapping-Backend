# # FarmMapping

# Tiling
Tiles are create using QGIS
- add raster layer, select basemap.
- processing tools widget on right side - generate XYZ tiles - 15-22 zoom
- Serve using spring itself for prototype. Just add to static resources folder.

# Icons and map
- Leaflet library handles display and overlap of icons.
- Observations endpoint provides Long and Lat cordinates of icon, and references image file.
